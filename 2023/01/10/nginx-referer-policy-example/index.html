Nginx referer 配置例子

Tue Jan 10 2023

网站需要配置防盗链，通过 Nginx 过滤非法的 referer

location ~ .*\.(png|jpg|...)$ {
    valid_referers none blocked server_names [domains];
    if ($invalid_referer) {
        return 403;
    }
}

一些说明：

  • server_names 代表当前 server 块中的所有 server_name
  • domains 可以使用泛域名
